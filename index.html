git commit -a -m ""

git commit - zatwierdza zmodyfikowane i usunięte pliki w danym repozytorium.
-a (all - wszystkie) - parametr, dzięki któremu zatwierdzone zostaną wszystkie zmiany.
-m (message - wiadomość) - parametr, którty oznacza dodanie do "commita" krótkiej wiadomości.
"" - miejsce na krótką wiadomość.
